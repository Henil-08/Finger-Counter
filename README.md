# Finger-Counter

Why use the traditional method to count the fingers when pointing your palm at the camera do it for you? This Code basically tracks 21 Points on ones hand using Mediapipe and openCV modules. Using that module one can count ones fingers by point the hand towards the camera.

## How to Finger-Counter:
1. Have a Hand
2. Use your Right Hand
3. Point it towards the camera
4. Fingers are displayed on bottom right
